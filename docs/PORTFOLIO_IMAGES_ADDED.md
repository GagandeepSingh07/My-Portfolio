## 💡 Pro Tips

### **Adding New Items:**
```javascript
{
    title: 'Your New Project Title',
    image: '/images/portfolio/category/filename.jpg',
    category: 'social-media', // or '3d-work', 'logos', 'reels', 'certificates'
    badge: 'new', // optional: 'new', 'recent', 'replaced', 'old'
    type: 'image', // or 'reel', 'landing'
    playIcon: true // only for video content
}
```

### **Badge Guidelines:**
- **NEW**: Just completed (within 1 month)
- **RECENT**: Completed within 3 months
- **No badge**: Older work still worth showing
- **REPLACED**: Updated version exists
- **OLD**: Historical reference

### **Image Optimization Tips:**
- Consider converting large PNG files to WebP for better performance
- Recommended max width: 1920px
- Keep file sizes under 500KB when possible
- Use JPG for photos, PNG for graphics/logos
