# Creative Portfolio – Graphic Designer & Video Editor

A modern, responsive portfolio website built with Next.js 16, featuring smooth animations and a stunning glassmorphism design.

## 🎨 Features

- **Responsive Design** - Looks great on all devices
- **Smooth Animations** - Powered by Framer Motion
- **Smooth Scrolling** - Enhanced UX with Lenis
- **Modern UI** - Glassmorphism effects and gradient backgrounds
- **Fast Performance** - Built with Next.js 16 and Turbopack
- **SEO Optimized** - Meta tags and semantic HTML

## 🚀 Tech Stack

- **Framework:** Next.js 16.0.10 with Turbopack
- **React:** 19.2.1
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion 12.23.25
- **Smooth Scrolling:** Lenis 1.3.11
- **Icons:** Lucide React

## 📦 Getting Started

1. **Clone the repository**
```bash
git clone https://github.com/yourusername/creative-portfolio.git
cd creative-portfolio
```

2. **Install dependencies**
```bash
npm install
```

3. **Run the development server**
```bash
npm run dev
```

4. **Open your browser**
Navigate to [http://localhost:3000](http://localhost:3000)

## 🛠️ Customization

### Update Content
Edit the content in each section file located in the `/sections` directory:
- `hero-section.jsx`
- `featured-clients.jsx`
- `services.jsx`
- `creative-process.jsx`
- `testimonials.jsx`
- `faq-section.jsx`
- `service-packages.jsx`
- `call-to-action.jsx`

### Update Branding
- Replace `/public/assets/logo.svg` with your logo
- Update colors in `/app/globals.css`
- Modify navbar and footer in `/components`

### Update Images
Replace the placeholder images in `/public/assets`:
- `workflow1.png`, `workflow2.png`, `workflow3.png` - Process images
- `company-logo-*.svg` - Client logos
